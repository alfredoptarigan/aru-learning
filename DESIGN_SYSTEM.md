# Pixel Art Design System

## Typography
**Font Family**: `VT323`, monospace, cursive.
- **Headings**: `VT323` (Bold/Regular depending on context).
- **Body**: `VT323`.

## Color Palette
| Name        | Hex Code  | Usage                        |
|-------------|-----------|------------------------------|
| **Primary** | `#C25E28` | Main actions, active states. |
| **Secondary**| `#4A90E2` | Secondary actions.           |
| **Accent**  | `#F1C40F` | Highlights, warnings.        |
| **Destructive**| `#E74C3C` | Errors, delete actions.      |
| **Background**| `#FDFBF7` | Page background.             |
| **Surface** | `#FFFFFF` | Card backgrounds.            |
| **Border**  | `#000000` | Borders, text.               |

## UI Elements

### General Rules
- **Corner Radius**: `0px` (Sharp corners).
- **Borders**: `2px` solid black (`#000000`).
- **Shadows**: `4px 4px 0px 0px #000000` (Hard offset).

### Buttons
- **Primary**: Primary Color bg, Black text, Thick Border, Hard Shadow. Active: Translate down-right, remove shadow.
- **Secondary**: Secondary Color bg, Black text, Thick Border, Hard Shadow.
- **Outline**: Transparent bg, Black Border, Hard Shadow.

### Inputs
- **Default**: White bg, Thick Border.
- **Focus**: Thick Outline (Ring), No glow.

### Cards
- **Style**: White bg, Thick Border, Hard Shadow.
